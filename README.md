Rest API Example
=====

Rest APIs by [realworld_api](https://realworld-docs.netlify.app/docs/specs/backend-specs/endpoints) implemented in Erlang

## Run

```
$ rebar3 shell
```

## Test

```
$ cd run\_api\_test
$ ./run\_api\_test.sh
```

## Files:

- `user\_h.erl`: POST, PUT, GET for creation, update and get users
- `profile\_h.erl`: POST
